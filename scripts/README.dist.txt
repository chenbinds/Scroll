卷轴 Scroll v{version} - Windows 便携版
================================

【使用方法】
1. 解压本文件夹（若收到的是 zip）
2. 双击 Scroll.exe 即可运行
3. 无需安装 Node.js 或其它依赖

【系统要求】
- Windows 10 / 11 (64 位)
- 首次启动可能需 5-10 秒，属正常现象

【数据存储】
阅读进度、书架、标注等保存在本机用户目录，卸载 exe 不会自动删除。

【常见问题】
- 杀毒软件误报: 右键 exe -> 属性 -> 解除锁定; 或将文件夹加入 Windows 安全中心「排除项」(详见 dist 包外 docs/06-troubleshooting.md)
- 无法启动: 确认系统是 64 位; 完全退出后重开

【说明】
本地电子书阅读器，支持 EPUB / PDF / MOBI / TXT / 漫画等格式。

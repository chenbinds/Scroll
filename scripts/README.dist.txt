卷轴 Scroll v{version} - Windows 便携版
================================

【使用方法】
1. 解压本文件夹（若收到的是 zip）
2. 双击 Scroll.exe 即可运行
3. 无需安装 Node.js 或其它依赖
4. 整夹复制到其他电脑即可运行（含 UserData 子目录时数据一并带走）

【系统要求】
- Windows 10 / 11 (64 位)
- 首次启动可能需 5-10 秒，属正常现象

【数据存储】
阅读进度、书架、标注等保存在本文件夹内的 UserData/ 子目录。
请勿只复制 Scroll.exe — 需保留整个文件夹（含 resources 等文件）。

【常见问题】
- 杀毒软件误报: 右键 exe -> 属性 -> 解除锁定; 或将文件夹加入 Windows 安全中心「排除项」(详见 dist 包外 docs/06-troubleshooting.md)
- 无法启动: 确认系统是 64 位; 完全退出后重开; 确认未只拷贝了 exe

【说明】
本地电子书阅读器，支持 EPUB / PDF / MOBI / TXT / 漫画等格式。

Scroll v{version} - Windows portable
====================================

1. Unzip this folder (if you received a zip)
2. Run Scroll.exe
3. No Node.js or other runtime required
4. Copy the whole folder to another PC to run (include UserData/ to keep your library)

Requires Windows 10/11 (64-bit). Supports EPUB, PDF, MOBI, TXT, comics.
Data is stored in UserData/ next to Scroll.exe.

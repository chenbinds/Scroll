# 音乐播放器 — 功能规格

> 版本: v0.1.0 | 日期: 2026-07-11 | **P3 实现：2026-07-14**（以代码 + `Handoff_音乐_2026-07-13.md` 为准）

---

## 功能概述

在阅读电子书时播放背景音乐。支持网络流媒体和本地文件。严格遵守版权规范。

## 功能清单

### 1. 迷你播放器 (Mini Player)

- **触发:** 首次播放曲目时自动显示在底部
- **位置:** 窗口底部，固定 48px 高度
- **显示内容:**
  - 当前曲目标题 + 艺术家
  - 播放/暂停按钮
  - 上一首/下一首
  - 音量滑块（隐藏在小屏）
  - 当前播放时间
  - 展开播放列表按钮
  - 关闭按钮
- **行为:** 阅读模式切换时保持播放状态

### 2. 完整播放列表面板

- **触发:** 点击迷你播放器上的展开按钮
- **显示:**
  - 所有曲目列表，当前播放曲目高亮
  - 每首曲目显示：标题、艺术家、风格标签、版权标识
  - 删除按钮
  - 底部操作：添加网络 URL、添加本地文件
  - 版权声明文字

### 3. 内置预置曲目

来自 Pixabay Music（免版权，可商用）：

| 曲目 | 风格 |
|------|------|
| Ambient Dream | ambient |
| Calm Waters | ambient |
| Lo-fi Chill | lofi |
| Peaceful Piano | classical |
| Gentle Rain | nature |

**注意:** 现行实现为 **Web Audio API 实时生成** 内置氛围音（见 `musicStore.ts` / `audioGenerator.ts`），非 Pixabay CDN。规格中 Pixabay 预置为早期方案，以代码为准。

### 4. 用户自行添加

#### 网络 URL
- 用户在弹窗中输入 URL
- 系统提示"请确保拥有合法使用权"
- 可以填写标题和艺术家信息

#### 本地文件
- 通过文件对话框选择本地音频文件
- 支持格式：MP3, WAV, FLAC, OGG, AAC, M4A, WMA
- 路径存储为绝对路径（本地播放用 `file://` 协议）

### 5. 版权声明

在播放列表面板底部始终显示：

> ⚠️ 版权声明：内置预置曲目来自 Pixabay Music（免版权，可商用）。
> 用户自行添加的 URL 和本地文件请确保拥有合法使用权。
> 本软件不提供侵权内容，也不存储任何音乐文件。

## 技术实现

### 音频播放

使用 HTML5 `<Audio>` 元素：
- 网络 URL：直接设 `src`
- 本地文件：`file:///` 协议
- 自动播放策略：用户首次交互后允许

### 状态管理

Zustand store (`musicStore.ts`)：
- `playlist`: 曲目列表
- `currentIndex`: 当前播放索引
- `isPlaying`, `volume`, `currentTime`, `duration`: 播放状态
- `showMiniPlayer`, `isExpanded`: UI 状态

### IPC 接口

- `dialog:openMusic` — 打开本地音频文件对话框

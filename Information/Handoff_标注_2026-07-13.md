## 架构红线（勿破）

1. **不改正文 DOM**；Overlay / HighlightLayer 与 `reader-scroll` 为兄弟层。
2. Canvas **仅视口大小**；坐标相对文档归一化 0~1。
3. 标注存 `settings_annotations_{bookId}.json`；`AZW/AZW3` → format=`mobi`。
4. **一条离开通道**：`requestLeave` → React 弹窗 → `confirmClose` / `cancelClose`。

## 格式映射

```
EPUB → epub
MOBI / AZW / AZW3 → mobi
PDF → pdf
```

## 关键接入点

- `EpubReader.tsx` / `MobiReader.tsx` / `PdfReader.tsx`
- `annotationFormatForBook()` / `markToolSupported()` in `annotationTypes.ts`
- PDF：`layoutKey={scale}` 保证缩放后重绘

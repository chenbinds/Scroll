## 7. 重要决策与约束（红线）

1. **MOBI/AZW3**：foliate-js 全量渲染；**禁止** Calibre / 自研二进制
2. **EPUB/MOBI**：全量 HTML 渲染；NCX 用 DOMParser，禁正则
3. **标注**：不改正文 DOM；视口兄弟 canvas；坐标 0~1；`annotations_{bookId}` 独立 JSON
4. **离开**：仅 `requestLeave` → React 弹窗；禁止 MessageBox 双轨
5. **封面**：`userData/covers/{id}.jpg`，JSON 只存 `scroll-cover://`
6. **豆瓣**：导入拉一次 + 手动刷新；公司网需错误反馈
7. **PowerShell**：用 `;`，不用 `&&`

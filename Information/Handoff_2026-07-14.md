## 7. 重要决策与约束

- MOBI → **foliate-js**；禁止 Calibre / 自研 MOBI 二进制解析
- 标注**不改正文 DOM**；离开唯一通道 `requestLeave` → React 弹窗
- 书签 `bookmarksByBook`；标注 `annotations_{bookId}.json`
- 查义：选区 **> 80 字不显示**按钮（用户确认够用，勿擅自放宽）
- 打包：portable、关 code signing；国内镜像
- `release.bak-20260713/` 勿入 git
- PowerShell 用 `;`，不用 `&&`

## 8. 已知坑

- 标注跳转：须直接 DOM 滚动，不能只更新 store 后指望自动滚
- 字号变后高亮须按 `textStart`/quote 重算，勿死缓存像素 rect
- EPUB `extractBody` 会剥掉 `<style>`，段首缩进靠全局 CSS（内联 `text-indent:0` 仍尊重）
- `pack.bat` 勿用 `npx` 裸调；README.dist 用 UTF-8 BOM 模板
- main/preload 变更后须**完全退出**再开 `Scroll.vbs`；分发须重新 `pack.bat`
- 杀软可能拖慢冷启动：见 `docs/06-troubleshooting.md` 加排除项

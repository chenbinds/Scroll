## 7. 重要决策与约束

- MOBI → **foliate-js**；禁止 Calibre
- 标注不改正文 DOM；离开仅 React 弹窗（`requestLeave`）
- 书签 `bookmarksByBook`；标注 `annotations_{bookId}`
- 音乐：内置 Web Audio generator；不打包侵权音频；本地/URL 只存路径
- 不集成 Listen 1（版权/维护成本；已否决）
- 打包：portable 单文件，关闭 code signing（避免 winCodeSign 权限/下载问题）

## 8. 已知坑

- `pack.bat` 勿用 `npx` 裸调（曾误入 `npm exec` 交互壳）；现已改 `node_modules\.bin\*.cmd`
